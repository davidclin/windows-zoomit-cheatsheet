# Zoomit

### Draw Shapes
```
Ctrl + 2        Draw mode
Box             Ctrl + mouse
Arrow           Ctrl + Shift
Freehand        Shift + mouse
Exit            Esc
```

### Colors
```
r              Red
o              Orange
y              Yellow
g              Green
b              Blue
p              Pink
```

### Zoom
```
Ctrl + 1        Mouse scroll to zoom in/out
```
