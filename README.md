# Zoomit

### Modes
```
Ctrl + 1       Zoom
Ctrl + 2       Draw
Ctrl + 3       Time       
Ctrl + 4       LiveZoom

```

### Draw Shapes
```
Ctrl + 2        Draw mode
Box             Ctrl + mouse
Arrow           Ctrl + Shift
Freehand        Shift + mouse
t               Type text
shift + t       Type text (right aligned)
Exit            Esc
```

### Colors
```
r              Red
o              Orange
y              Yellow
g              Green
b              Blue
p              Pink
```

### Zoom
```
Ctrl + 1        Mouse scroll to zoom in/out
```

### Timer
```
Ctrl + 3        Set time in settings
```
